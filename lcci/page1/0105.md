# 面试题 01.05. 一次编辑

## [题目](https://leetcode.cn/problems/one-away-lcci/)
字符串有三种编辑操作:插入一个英文字符、删除一个英文字符或者替换一个英文字符。 给定两个字符串，编写一个函数判定它们是否只需要一次(或者零次)编辑。

**示例 1:**

```
输入: 
first = "pale"
second = "ple"
输出: True
```

**示例 2:**

```
输入: 
first = "pales"
second = "pal"
输出: False
```
