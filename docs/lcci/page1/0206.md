## [题目](https://leetcode.cn/problems/palindrome-linked-list-lcci/)
编写一个函数，检查输入的链表是否是回文的。

**示例 1：**

```
输入： 1->2
输出： false 
```

**示例 2：**

```
输入： 1->2->2->1
输出： true 
```

**进阶：**   

你能否用 O(n) 时间复杂度和 O(1) 空间复杂度解决此题？
