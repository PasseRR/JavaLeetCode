# 175. 组合两个表
## [题目](https://leetcode.cn/problems/combine-two-tables){:target="_blank"}

表1: Person

| 列名         | 类型      |
|:-----------|:--------|
| `PersonId` | int     |
| FirstName  | varchar |
| LastName   | varchar |

表2: Address

| 列名          | 类型      |
|:------------|:--------|
| `AddressId` | int     |
| PersonId    | int     |
| City        | varchar |
| State       | varchar |

编写一个 SQL 查询，满足条件：无论 person 是否有地址信息，都需要基于上述两表提供person 的以下信息：

> FirstName, LastName, City, State
