### [题目](https://leetcode.cn/kth-node-from-end-of-list-lcci/){:target="_blank"}

实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。

注意：本题相对原题稍作改动

示例：  
输入： 1->2->3->4->5 和 k = 2  
输出： 4

说明：  
给定的 k保证是有效的。

### 题解

```java
public int kthToLast(ListNode head, int k) {
    int count = 0;
    ListNode node = head;
    // 计算链表长度
    while (node != null) {
        count++;
        node = node.next;
    }

    int i = 0;
    node = head;
    // 找到倒数k位置的数字
    while (node != null) {
        if (i + k == count) {
            return node.val;
        }
        node = node.next;
        i++;
    }

    return 0;
}
```
